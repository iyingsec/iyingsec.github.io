登录到vps

使用Xshell等工具登录你的vps

更新

#apt-get update

安装所需环境Apache2 mysql php5以及扩展

安装Apeche2#apt-get install apache2
安装MySQL#apt-get install mysql-server密码需要记住！
安装PHP5#apt-get install php5

继续安装扩展#apt-get install php5-mysql 否则会出现（您的 PHP 似乎没有安装运行 WordPress 所必需的 MySQL 扩展错误）

下载WordPress,并解压

#cd /var/www/html
#wagt https://cn.wordpress.org/wordpress-4.3-zh_CN.tar.gz

解压tar.gz #tar xvfz wordpress-4.3-zh_CN.tar.gz
如果你下载的是zip,那就使用#unzip wordpress-4.3-zh_CN.tar.gz

进入WordPress编辑配置

#cd /var/www/html/wordpress
编辑配置#vi wp-config-sample.php

/ MySQL数据库名 */
define(‘DB_NAME’, ‘wordpress’);
/ MySQL数据库用户名 /
define(‘DB_USER’, ‘root’);
/** MySQL数据库密码 /
define(‘DB_PASSWORD’, 密码’)
/* MySQL主机（保持原样就行） /
define(‘DB_HOST’, ‘localhost’);

修改完后重名为wp-config
#mv wp-config-sample.php wp-config.php

重启Apeche和MySQL

重启Apeche2#/etc/init.d/apache2 restart
重启MySQL#/etc/init.d/mysqld restart

新建数据库并修改mysql权限

修改文件路径权限#chomd 777 /var/lib/mysql
#mysql -u root -p输入密码
新建数据库mysql>create database wordpress;

新增WWW的数据写入权限

#chown -R www-data:www-data /var/www/html/wordpress

解析域名

现在Apeche2不依靠httpd.conf来配置。
Ubuntu下默认的配置文件路径在/etc/apache2/sites-available/default
进入配置路径#cd /etc/apeche2/sites-available
配置#vi 000-default.conf
修改文件路径为/var/www/html/wordpress/
然后把域名解析到你的vps提供商提供给你的ip

安装WordPress

访问你的域名，进行5分钟安装WordPress
