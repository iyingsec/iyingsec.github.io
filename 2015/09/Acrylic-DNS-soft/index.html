下载Acrylic DNS

点击进入Acrylic DNS官方并下载

安装Acrylic DNS

双击Acrylic，一直下一步就行。

配置

进入C:\Program Files(x86)\Acrylic DNS Proxy
打开AcrylicConfiguration
在TertiaryServerAddress后添加DNS
