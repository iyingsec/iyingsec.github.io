Ubuntu太卡，怎么办？

2015-01-28 | 分类于 折腾笔记

请原谅不会写标题的我,大内存的用户请直接略过……

初中那会用过一年的Linux,在那一年,可把我折腾死了.
记得刚开始安装好了,联网都折腾了好几天……
也没学到什么东西,就换回Window了
以前用的是Ubuntu Kylin,运行的很流畅.
这次换成了Ubuntu,安装完后感觉太吃内存了.
本来都打算换其他发行版的了,
无聊逛Ubuntu中文论坛看到一个帖子
一条命令，令你的1G内存变2G
用CPU换内存(可能是吧……)解决了太吃内存而变卡的问题

首先运行sudo apt-get install zram-config
然后重启之
注意：2G内存以下效果非常明显，2G内存 ～ 4G内存效果一般，4G内存以上，基本无感觉。

介绍：http://www.webupd8.org/2011/10/increased-performance-in-linux-with.html
效果：http://www.ubuntu-tw.org/modules/newbb/viewtopic.php?topic_id=57168
用法（我自己用的就是默认的，完全没管过它）：http://askubuntu.com/questions/174579/how-do-i-use-zram
