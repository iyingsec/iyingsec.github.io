不作就不会活

发表于 2015-12-20   |   分类于 所思所想

她说，“作”就是不安现状的瞎折腾。她说，不想要那种循规滔规、安安稳稳、平平淡淡的日子，不想要那种一眼就可以看到死的那种人生。她说，一个没有把白酒都尝遍的人，他是不大懂的清水之味的。一个一辈子都安分守己不敢“作”的人，他从来也不曾拥有一个精彩丰富的人生。毕竟，no zuo, no die，就no life!

那个她就是北大才女，安徽卫视《超级演说家》第二季的冠军——刘媛媛，开篇引用了她的演讲《不作就不会活》。为什么要发这样一篇文章？因为她说的话触动了我，或者说是触动了我们这些爱折腾的年轻人。但是，我们不安于现状的折腾又是因为什么呢？是想要一个精彩的人生吗？我不知道其他人是怎么想的，反正我折腾纯属因为兴趣，之所以不满足于现状，是自己没那个能力享受想要的生活。

以前是折腾些小玩意，弄个博客，翻翻墙，复制个代码装B……. 随着年龄的增长，我们的舞台变的更大了。我们可以创建一个公司，或者一个团队、乃至一款创新产品，只因自己喜欢。能不能做大、做到你们眼中的成功其实无所谓，重要是经历就好。说到底，折腾不是瞎折腾。而是有目的的折腾，其实说白了就是做自己喜欢做的事，不要在意其他人的眼光。

我也一直在我们宿舍灌输“人生重要的是经历”，可到底什么才是经历，就连我自己都不知道。我只知道，在我的想象中，有一个沧桑的少年，讲着他的故事。。。好了，不扯了。最后附上网络一段子与君共勉！

“有人说生命在于静止，有人说生命在于折腾……其实，懒有懒的安逸，折腾有折腾的乐趣。重要的是，你有享受过程的智慧，又有承担后果的能力。” ——来源网络，出处不详。

我们不可能看遍所有的风景

发表于 2015-10-24   |   分类于 所思所想

我们不可能看遍所有的风景，学会放下，也是一种释然。

最傻逼的不是不拼搏，而是不知道在为什么拼搏。混完了初中，混中专。起初选择电子专业是为了单片机，第一年老师告诉我，今天先学基础，第二年才开始学专业知识，也就是所谓的单片机。可到了第二年，单片机老师病假。打着“多学一点没啥不好”的名义，教着我们机械系的钳工，其实是为了增加就业率罢了。前一段时间，我还一直以为有个所谓的“单片机大赛”，所以就问合班后的副班长借了本51单片机的书，啃了大约几十页，毫无卵用。唯一觉得这本书没有白看的原因只有一段话。如下：

转眼就到了本科毕业聚会了，大部分同学的大学生活就此画上了句号。我亲眼看着他们如何一步步地成就了自己的梦想，也亲眼看着他们如何一次次地重复着自己的生活，同时也亲眼看着他们如何一天天地堕落下去。从那天起，大家又将迎来各自的全新环境，在那里，也许有人庆幸、也许有人后悔，但无论怎样，自己的路自己选择，当然要自己走下去。
大学生活是我们人生中最宝贵的经历，我们付出了四年光阴的代价，我们应该也必须为此有所收获。很多同学在大学校园里迷失了方向，不知道自己来到这里究竟是为了什么，单纯的认为毕业后找个好工作就万事大吉了，可你们是否想过，没有真正的本领如何能找到好工作？如何能为这个社会做出贡献？更现实一些，没有一份好工作如何买房成家，建立起属于你自己的幸福快乐的家呢？真本事不是靠混日子混出来的，我们身边有很多机会可以发挥自己的能力，实现自己的理想，即使这些机会不是很明显地让我们看见，我们也应该努力地去争取。偶尔的娱乐是可以的，可是成宿在网络游戏中挥舞大刀的同学们，那里可以砍出你的未来吗？适当的运动也是应该的，可整天都泡在篮球场上的同学们，你觉得你还有希望成为第二个乔丹吗？哥们义气固然重要，可三天两头为朋友两肋插刀，你的肋骨够用吗？年轻人应该有更高的追求，你要为你的将来做好准备，外面的世界很精彩，外面的世界也很残酷，你活着的每一天都应该好好珍惜！

而所谓的“单片机大赛”其实正确的叫法应该叫“电子设计大赛”，简单的了解了一下，基本上都是以大学学校组建的团队参赛，而我们技校根本就不用想了。想前想后，还是放弃继续学习单片机。原因有二，一是因为我的基础确实太差，从入学到现在根本没有学习过。就如合班之后的副班长说的那样，你知道什么是按钮吗？连按钮是啥都不知道，还想去培训学习单片机？虽然知道是用来按的，但还说不出来按钮到底是个啥。翻翻课本，才知道按钮是控制电器元件，常用来接通或断开‘控制电路’。这些最最最基础的东西我都不会，我有什么资格去培训学习！二是因为单片机后期嵌入式开发需要的基础太难了，门栏略高。所以我选择了放弃，不是因为惧怕太难，而是选择了更好的选择！也许以后会整装再战嵌入式开发！不管怎样，我是以过好这一生为目标。所以，我不可能去尝试所有喜欢的东西。

本文也算是对 我放弃学习单片机 的一个交代了。

路还长，还能精彩起来！

其实，我每天都不在状态。虽然在熟人面前很活泼，但对不熟悉的人犹如一座冰山。

关于博客，我有话要说。

发表于 2015-10-05   |   分类于 所思所想

忙碌的生活，已经使我忘记了曾经的梦想。可这种忙碌只是瞎忙，毫无价值。静下心来，好好想想我所困扰的问题。（由于原文惨不忍睹，加上当时心情不好，所以只是随便写了写。本文已于2015年11月7日重新编辑发布，愿标题：谈谈博客的价值（ps：有点高大上,跑题了，所以换掉了））

“你所处的圈子，决定你的人生高度。”

经常听到有人说“你混的圈子毫无价值,瞎混”云云…可怎么知道圈子的价值呢？我觉得每一个圈子都有他所存在的价值，搞技术的瞧不起搞娱乐的，搞娱乐的瞧不起搞技术的。各有各的优点，各有各的价值。之所以不被互相认可，是因为对你没价值而已。

我为什么要写博客、到什么时候关掉博客？

为什么要写博客？这个问题想了很多遍，也想出了很多答案。最有说服力的一句答案是“做自己喜欢做的事”，是啊，做自己喜欢做的事情才会不断的有动力。至于为什么要写博客，自然也有了答案。也许是为了记录，看看曾经的幼稚，或者看看现在的自己与以前的我的变化。也许是为了分享，分享自己的收获，帮助别人解决困难。也许是为了炫耀，炫耀身边的人没有，只有自己有。也许是把Blog当作一份简历，但似乎我的博客技术文很少。之所以要公开显示，而不是私密。是因为我想通过独立博客的形式结交志同道合的朋友，也许这项任务社交网络更适合，但我想独立博客应该也不会差。建独立博客是为了无限制，所以能更好的进行思想交流。在这个浮躁的世界里，能偶尔静下心来思考思考，写点文字。真好！

有始有终，天下没有不散的宴席。我不可能花费很多年的时间去维护这个博客，我也没那么多精力。说不定那天我的博客就在互联网中消失了。有生之年，如果没有拼到想要的成就，也许还会重开这个博客。也只是也许，毕竟未来还是个未知。

谈谈”友情链接“

有时候翻翻以前的文章，点击以前来过我这里的博主的博客链接，发现很多链接都打不开了，有的连域名都过期了。就这样悄无声息的离开了，难免有一丝伤感。关于友情链接，以后只换自己喜欢的，欢迎申请，也许你是我的下一个朋友。

Hexo 更换电脑后

发表于 2015-10-01   |   分类于 折腾笔记

起因：因下载了一个ipv6代理软件，强迫症严重发作。
所以，我必须重装电脑！！！
更换电脑或者重装电脑后，Hexo该这样…

备份

备份hexo目录下的source目录（文章）、_config.yml文件（配置）、themes文件夹（主题）。
备份完成后，就可以重装电脑啦~

安装必备软件

nodejs
git

安装Hexo以及依赖环境

npm install -g hexo-cli -g
cd 到 hexo 目录键入$hexo init
然后npm install安装依赖环境
之后就可以$hexo server看到效果了..

安装插件

npm install hexo-deployer-git --save 部署
npm install hexo-generator-feed --save 订阅
npm install hexo-generator-sitemap --save 网站地图
以上是我所使用的插件，其他插件自行搜索。

还原备份

直接把备份文件替换掉新的就可以了

密钥上传

可以参考最下边的链接生成密钥，我用的是Github Desktop直接生成密钥并上传。
所以我只需更改下Hexo所在的目录就可以了。

上传

然后就可以$hexo d -g部署了
如果无法上传，可以替换成git 或者 https
