谈改变

发表于 2015-05-01   |   分类于 东扯西扯   |  

人都不是十全十美的，总会有几个缺点。而那几个缺点中，总会有几个自己认为不是缺点的缺点。大多数人们都想改掉那个缺点，最后却只有少数人能如愿以偿。而我，荣幸的成为了“大多数人”。究其原因，我想是 1.给自己留太多的后路，没有控制好娱乐时间和学习时间 2.忘记了为什么出发，为什么要做这件事。 在这篇文章中，我不想谈我的改变经历以及结果。首先，对他人没有价值可言。其次，一次又一次的失败，我还在改变、在成长的路上…

周围的人，对自身的影响起着很多的作用。近朱者赤，近墨者黑。你和什么样的人在一起，就会拥有什么样的人生。所以，如果你对现在的环境、身边的人感到不满意，那就去试着改变周围的人吧！这不是浪费时间，在培养说服力的同时，也潜意识的在改变身边的环境。但我更建议你，远离你觉得没有价值的人。理由是，没有必要浪费自己的时间去教育一个傻X。可能这句话有点过激，但事实就是如此，至少我现在很赞同。

改变，很难。我一个中专生已经比同龄的高中生拉下一大截了，就学习时间而言，说是中专生的二三倍也不为过。有人说，技校生比大学生早踏入社会，大学生不如技校生什么的。甚至还有一个故事，A是学习不好的差生，B是学习优秀的好学生。中考后，A踏入社会，B顺利的考上了理想的高中。后来，A开了一家公司，B上完了大学继而进入了一家公司当经历，而那家公司的老板就是A。可能还有很多这样的故事，但你们有想过A的家境么？说不定他本就是富二代呢。就算是白手起家，你们有考虑过比例问题么？还能有多少像A一样的差生？故事总是美好的，现实告诉我们不努力还是不行的。

你要做的就是，尽可能地让自己更优秀。这样你才可以跟所谓的“少数人”拼运气！

Fedora 安装fcitx输入法

发表于 2015-04-05   |   分类于 折腾笔记   |  

之前用的Linux都是debian系的，换成fedore居然发现差距好大..不过用着用着又发现基本都是相同的…
原文地址，我只是在解决问题的同时充当了搬运工…

执行这三条命令

yum安装命令

  • sudo yum install fcitx-pinyin
  • sudo yum install fcitx-configtool
  • sudo yum install im-chooser
    如果你忘记安装fcitx-configtool，你的系统中也是有这个命令的，但是调用它时会报错：

您正在图形界面下运行 fcitx，但是 fcitx-configtool 未被找到，软件包名通常为 fcitx-config-gtk，fcitx-config-gtk3，或者 fcitx-config

所以，请务安装fcitx-configtool。
gsetting配置

按说，这时候该使用 im-chooser 选择输入法为fcitx了。如果你选择fcitx后报错显示：

GDBus.Error:org.gtk.GDBus.UnmappedGError.Quark.imsettings2derror_2dquark.Code5: Current desktop isn’t targeted by IMSettings.

查看日志，关键一句为：

INFO: Attempting to switch IM to FCITX [lang=en_US.utf8, update=true] org.gnome.settings-daemon.plugins.keyboard.active is true. imsettings is going to be disabled.

这就需要对gsetting设定：

gsettings set org.gnome.settings-daemon.plugins.keyboard active false

现在再去设定im-chooser中选择fcitx，重启后就可以使用了。
fcitx配置

fcitx默认的快捷键比较诡异，使用 fcitx-configtool 进行配置。

在全局配置的快件键->显示高级选项中，我把切换激活/非激活输入法设定为左shift，输入法切换键设定为Ctrl+Shift。

推荐一首歌 活着-郝云

最近因为种种原因，格外的坠落。想过要退学，但最后还是放弃了退学的想法。

为什么要退学？因为一个人，又因为一件事。我感觉我天生就不是学习的料，用实话说就是我从没认真坚持学习过、认真过。也许只是在小学初期阶段认真学习过吧，依然记得那时候一二年级数学都是90分以上。三年级的时候忘记了成绩怎样，但家里有张三年级时候发的“三好学生”奖状。大约从四年级开始，我就开始玩游戏。渐渐地不爱学习，然后到了初中，以第46名(当时班里大约有60人)成绩继续往下沉，直到八年级沉到倒数第一。来到技校后，第一学期的成绩我又是倒数第一。班里平均每人的成绩都在400以上，仅有少数几个300+分的和我一个295.4分的傻逼。

5个问题，5个解决方案。世界上没有最完美，也望提出你的一点建议。也许会对我、对读者有一定的帮助!
-1.第二学期开学的时候,我才知道专业课课本都学完了.我以前甚至都没去看过一页…
+1.坚持跟着老师的思路听完每一节课(每天听完的课都用纸记录下来)
-2.态度问题
+2.认真对待每一件事,哪怕微不足道的小事也一样.
-3.娱乐时间过长,形成惰性.
+3.停用QQ,每星期只能用5次.(用纸记录)
-4.杂念太多，安静不下来学习。
+4.专注于当下,不爱你的人你干嘛要这么在乎她.铭记3月1日晚上xx的一次教导,培养闪光点.
-5.学习气氛
+5.一:尝试孤独 二:在宿舍培养气氛

我不知道我现在走的路是否正确

发表于 2015-02-28   |   分类于 东扯西扯   |  

1.写博客的意义

去年，我建立了我人生中的第一个独立域名的博客。当时每天都转载好几篇文章，觉得很有意思。不过没多久，就开始迷茫了。接着我开始写原创文章，以练习写作水平为由。主要都是关于改变的文章，而且不是如何改变！根本没有认真去写，全都是一气呵成的。还有一些写的是关于我现实中的故事，本来故事很美好，被我这么一写，整的跟2b似的。

每个人写博客的意义都是不同的…

  • 我觉得每个人都有自己写博客的初衷，在我看来，其实我部关心有多少人看了我博客或者我写的到底有没有意义，我就仅仅是想在这个浮躁的生活中记录点什么。————ma2ge
  • 对于我来说 博客就是个个人地盘 想写什么写什么 不在乎有没有人看————慢灵魂
  • 博文的意义，我觉得至少有二：记录自己真实生活供老来回味以免大脑白痴，与博友交流思想共同提高。————Julius
  • 反正我是没有那么多理由，想折腾就折腾一下呗。————an9
  • 话说我的博客挺不正规的，都写的乱七八糟的东西 ————小二苘
    而我现在写博客，就是做我想做的事情这么简单。

2.收藏夹的书签过多,分不出哪些是有用的,该如何处理.

以前纠结浏览器的选择，后来慢慢地让自己接受“适合自己的就是最好的”，才渐渐放弃纠结这个问题。现在，纠结收藏的页面是否删除。这是拖延症患者常有的“毛病”，看到好的文章收藏之，然后..然后一放就是好几年。等到再想看的时候，又觉得太多，反而静不下心来看。想了想，决定还是以学习为主要目标，等我能静下心来的时候再拿出来看看。但是，强迫症的我忍不了啊！！忍不了也得忍！

3.一直想不明白的事

为什么自己始终没有改变？在初中的时候，从八年级最后一次考试直到中考，我的成绩都是班里倒数第一（自爆黑历史,表黑我!）为什么很多和我一样上课不听课的同学比我的成绩好？一直想不明白原因，可能是因为他们回家之后被父母逼着看了会书。或者抄的好、或者基础学的好…也可能是我一学期就学那么一二天的缘故吧…既然现在想不明白，那就留个以后去理解。

4.关于最近的学习情况

据统计，2015年2月12日-2015年2月28日，总学习时间为10小时左右。
为什么这么少，主要原因1.计划不完善 2.忍不住刷qq空间和微博..

5.接下来

马上就要开学了,开学后.不再以「以后工作不用所学的专业知识」为由而不学习.

PS：写这些东西,如果被搞安全的看到,肯定会说我是混娱乐圈的..不过没关系,反正我本来就没什么技术.还可能被一些大人看到,觉得我幼稚.不管怎么样,还是要发出来,因为我觉得他对我有帮助.

2014年之年底总结

发表于 2014-12-28   |   分类于 生活日记   |  

长这么大还是第一次认真的写总结，算是对今年做一个回顾吧！这篇总结也算是我小学作文水平过渡到初中水平的一个见证吧！除此之外，还要向大家炫耀一个事，因为最近右手受伤的缘故，吃饭玩电脑连洗头都是用一个手完成的！求表扬！哈哈^_^

今年年初，我还是一个在初中混日子的“差生”。记得那时，每天到学校就是睡觉、扰乱课堂秩序，中午吃完了饭，继续回到教室睡觉。年复一年，每天都是那样，终于熬到了6月份中考。在考场也是那样，不是睡觉就是东看看西看看，考完了，公布成绩了。和我耍的不错的几个小伙伴都考上高中了（毕竟都是校队的体育生嘛），而我只能另选道路，本来准备去学计算机编程的，但是当时的思想是拿到高学历才是最重要的，所以就去了技校，选择了一个和自己爱好还算接近的专业。

到了技校，报到完了，领了被子就去了宿舍。接下来就是军训了，军训没什么好写的，就一个字——“累”。大约军训了二个星期，结束后就是星期天了，然后收拾东西就回家了。再回到学校的时候，就开始上课了。在技校的这段日子里，也是天天睡觉，直到遇见她，这个习惯才开始慢慢好转。在第一个学期经历了很多事，有精彩，也有烦恼，总之，生活还得继续…

2014年12月17日，在教室和同学打聊，右手不幸划伤，可能是砸在暖气片上了，但我当时并没感觉到痛，过后才发现右手裂开一处伤口漏出了骨头，但那时已经是中午了，学校的医务室也关门了，班主任也不在，就只好先回宿舍简单的包扎一下，就去学校医务室等着开门了，终于等到医生来了，医生看看了问，什么时候弄的？我回到“上午”，然后医生就让我直接去医院缝。我没有班主任的手机号码，就只好上操场找同学要号码（我们班那时在上体育课），跟班主任说了之后，班主任给我开了假条，借了200元就去医院了。到了医院，医生也是问了我什么时候弄的，然后让我先去拍个片子看看，医生看看了没什么事，就跟我说，待会给你看看伤没伤着筋，要是伤着的话就得做手术了，没伤着给你缝下就可以了。直到这时，我才意识到严重性。最后右手缝了三针，请了假，只能明年再去上学了。（在这里也要提醒大家，冬天少打闹，容易受伤。注意保暖，别感冒了）

看到这里，你是否发现我并没有写学习的事？因为我今年，本来就没学到什么，还是在虚度光阴。中考过后的“长假”，我制定了一个计划，早已忘记，直到最近才想起来，但时光已飞逝，转眼就是2015年了。在2015年，可不能再继续这样了！
